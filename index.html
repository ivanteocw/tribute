Guido van Rossum




Time line of Van Rossum's life

  • 1956 - He was born in Haarlem, Netherlands.


  • 1982 - He received a master's degree in mathematics and computer science from the University of Amsterdam.


  • 1986 - While working at the Centrum Wiskunde & Informatica (CWI), Van Rossum wrote and contributed a glob() routine to BSD Unix in 1986 and helped develop the ABC programming language.


  • 1989 - Van Rossum had been looking for a "hobby programming project" when he decided to write an interpreter for a "new scripting language [he] had been thinking about lately".

  • 1991 - Python was first released. Python's design philosophy emphasizes code readability with its notable use of significant whitespace.

  • 1999 - He submitted a proposal to DARPA called "Computer Programming for Everybody", where he defined his goals for Python:

    1. An easy and intuitive language just as powerful as major competitors Open source, so anyone can contribute to its development
    2. Code that is as understandable as plain English
    3. Suitability for everyday tasks, allowing for short development times

  • 2000 - Python 2.0 was released, joined Zope corporation

  • 2003 - Left Zope corporation for Elemental Security. While there he worked on a custom programming language for the organization

  • 2005 - He worked at Google, where he spent half of his time developing the Python language

  • 2008 - Python 3.0 was released was a major revision of the language that is not completely backward-compatible, and much Python 2 code does not run unmodified on Python 3

  • 2013 - Left Google and started working for Dropbox.

  • 2019 - Left Dropbox and officially retired.
